# Make Contact Sheet

A Python script to create a rendering of a traditional film contact sheet from a folder of images.

## Requirements

This project requires Python 3.11 or newer.

Python packages are listed in `requirements.txt`:

```bash
pip install -r requirements.txt
```

The main third-party dependencies are:

- `opencv-python`
- `Pillow`

## Installation

Clone the repository:

```bash
git clone https://github.com/csalvaggio/make_contact_sheet.git
```

Create and activate a virtual environment:

```bash
cd make_contact_sheet
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install --upgrade pip
pip install -r requirements.txt
```
